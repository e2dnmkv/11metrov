# 11 метров - Комплекс мини-футбольных полей

Современный веб-сайт для футбольного комплекса.

## Технологии

- **React 18** + **TypeScript**
- **Vite** - быстрая сборка
- **Tailwind CSS** - стилизация
- **shadcn/ui** - UI компоненты
- **GSAP** - профессиональные анимации
- **React Router** - навигация
